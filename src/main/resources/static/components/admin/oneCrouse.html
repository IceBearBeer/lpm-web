基本人体生物学：细胞和组织




教师：Rachel Gibson、Catherine Snelling、Femke Buisman-Pijlman、Mario Ricci、Sophie Karanicolas


开课时间：2016-10-15 00:00:00--6-12-15 23:30:00


简介

你是否准备从事与健康相关的职业，或是想要学习健康科学？或许你是想要了解更多的人体的奥妙？我们的身体是奇妙而复杂的生物机器。本门课程会为你学习人体解剖学和生理学打下良好的基础。

你会更全面地了解人体细胞和组织在健康或疾病状态下的正常结构与功能。在课程中也会清晰阐述身体四种主要类型组织：上皮细胞，结缔组织，肌肉细胞和神经细胞。

本课程采用多种有趣的学习方式，包括短视频、动画、互动和综合测验。课程通过对核心概念进行引导，使你进行相关学习。

你将学到：

● 人体细胞和组织的结构和功能知识

● 基本人体解剖和生理学知识

● 健康科学以及健康职业的知识准备
